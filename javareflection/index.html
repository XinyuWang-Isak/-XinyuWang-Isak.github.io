Use setAccessible() to improve the effciency of reflection

2021-07-22
  • Method and Field, Construtor objects all have setAccessible() method.
  • setAccessible is a switch to change accessbility of security check. (identifier)
  • Set the parameter as true to cancel the Java language accessbility check.
  • 1. Improve the reflection effciency. If there are reflections in codes and need to be called frequently, please set parameter as true.
  • 2. Can access the attributes that identifier is private.

Comparison of execute speed.

//Normal way to call
    public static void testNormal(){
        User user = new User();

        long startTime = System.currentTimeMillis();

        for (int i = 0; i < 100000000; i++) {
            user.getId();
        }

        long endTime = System.currentTimeMillis();
        System.out.println("Normal way to execute 1 billion times: "
                + (endTime-startTime) + "ms.");
    }

    //Use reflection
    public static void testReflection() throws NoSuchMethodException, InvocationTargetException, IllegalAccessException {

        User user = new User();
        Class c1 = user.getClass();
        Method getId = c1.getDeclaredMethod("getId", null);

        long startTime = System.currentTimeMillis();

        for (int i = 0; i < 100000000; i++) {
            getId.invoke(user, null);
        }

        long endTime = System.currentTimeMillis();
        System.out.println("Reflection way to execute 1 billion times: "
                + (endTime-startTime) + "ms.");

    }

      //Turn off detection
    public static void testSetAccessible() throws NoSuchMethodException, InvocationTargetException, IllegalAccessException {

        User user = new User();
        Class c1 = user.getClass();

        Method getId = c1.getDeclaredMethod("getId", null);
        getId.setAccessible(true);

        long startTime = System.currentTimeMillis();

        for (int i = 0; i < 100000000; i++) {
            getId.invoke(user, null);
        }

        long endTime = System.currentTimeMillis();
        System.out.println("Turn off detection to execute 1 billion times: "
                + (endTime-startTime) + "ms.");

    }

Execution result of three methods:

Normal way to execute 1 billion times: 8ms.
Reflection way to execute 1 billion times: 712ms.
Turn off detection to execute 1 billion times: 314ms.
